    * 安裝 [Pandoc](https://pandoc.org/) : 功能強的萬用文件轉換器
    * 安裝 [MiKTeX](https://miktex.org/) : 在 Pandoc 轉檔時會自動安裝缺少的套件 (例如處理中文字就需要 cjk 套件)

* ### 輸出 PDF 步驟 (方法一)

    使用預設格式輸出，最簡單，輸出的內容格式也不會太美

    1. markdown 的 YAML 內宣告的內容會輸出成文件的封面

        ```yaml
        ---
        title: 使用 Pandoc 將 Markdown 轉為 PDF 文件
        author: Sam Chuang
        date: 2020-01-13
        ---
        ```

    2. 如果 markdown 文件內有中文字，要在開頭的 YAML 內加入中文字型宣告

        <b>`for Linux`</b>

        ```yaml
        ---
        mainfont: Microsoft YaHei Mono  
        ---
        ```

        <b>`for Windows`</b>

        ```yaml
        ---
        CJKmainfont: Microsoft YaHei Mono  
        ---
        ```

    3. 在 Terminal 或 PowerShell 或 CMD 輸入 Pandoc 指令

        <b>`for Linux`</b>

        ```PowerShell
        pandoc 'source.md' -o 'destination.pdf' --latex-engine=xelatex --toc
        ```

        <b>`for Windows`</b>

        ```PowerShell
        pandoc 'source.md' -o 'destination.pdf' --pdf-engine=xelatex --toc
        ```

        **參數說明**

        * `-o` <span style="color:#E3436B">'destination.pdf'</span> : 輸出檔名
        * `--latex-engine=xelatex` 或 `--pdf-engine=xelatex` : 指定 LaTeX engine 用 xelatex 做為 PDF 的編碼器
        * `--toc` : 自動在文件內產生 Table of Contents 目錄頁，並自動加上頁碼和連結 **(選擇性參數)**

    4. 範例檔參考 : Markdown 與輸出結果 PDF
