使用 Pandoc 將 Markdown 轉為 PDF 文件

發表於 2020-01-13 更新於 2020-01-14 分類於 Markdown Facebook： 0

  • 準備環境

    • 安裝 Pandoc : 功能強的萬用文件轉換器
    • 安裝 MiKTeX : 在 Pandoc 轉檔時會自動安裝缺少的套件 (例如處理中文字就需要 cjk 套件)

  • 輸出 PDF 步驟 (方法一)

    使用預設格式輸出，最簡單，輸出的內容格式也不會太美

    1. 使用 YAML metedata

      markdown 開頭的 YAML metadata 內宣告的內容會輸出成文件的首頁

    2. 指定中文字型

      如果 markdown 文件內有中文字，也要在 YAML metadata 內指定中文字型


      for Linux

      for Windows

    3. 輸入 Pandoc 轉檔指令

      在 Terminal 或 PowerShell 或 CMD 輸入 Pandoc 指令

      for Linux

      for Windows

      參數說明

      • -o 'destination.pdf' : 輸出檔名
      • --latex-engine=xelatex 或 --pdf-engine=xelatex : 指定 LaTeX engine 用 xelatex 做為 PDF 的編碼器
      • --toc : 在文件內產生 Table of Contents 目錄頁，並加上頁碼和連結 (選擇性參數)

    4. 轉檔指令 與 範例檔

      pandoc "example.txt" -o "example.pdf" –pdf-engine=xelatex –toc

      從輸出結果 example.pdf 來看，雖然轉換出來的格式不完美，但已可將中文內容轉出，也有 Table of Contents 目錄頁含頁碼與連結、以及文章大標題的書籤連結。

      example.txt 內的 YAML metadata 內容會被網站平台過濾掉，在原檔是存在的
      請參考 2. 指定中文字型 的 YAML metadata 內容

  • 輸出 PDF 步驟 (方法二)

    製作 LaTeX 檔指定為 template 格式輸出，輸出的格式最完美，也是最費工的做法，我目前的文章內容還沒多到需要製成這麼精致的 PDF，所以先打住，等將來需要的時候再繼續研究，或是您有使用的經驗，也請不吝留言分享，感謝您
